# Contributing to Saba-AgenticFlow

Thank you for your interest in contributing!

## Guidelines

- Saba-AgenticFlow is documentation-only (no functional code to implement)
- Contributions should add value to orchestration know-how
- All changes must pass CI (markdown lint, structure validation)
- Use clear, concise language

## How to Contribute

1. Fork repository
2. Create a branch for your feature
3. Make your changes
4. Run `./scripts/validate_structure.sh`
5. Submit a pull request

## Areas for Contribution

- New patterns and use cases
- Improved documentation and examples
- Better structure validation
- Bug fixes in existing patterns
